EduBid Insight Personal v1.0
================================

Education Sales Intelligence Platform

START
-----
Run EduBidInsight.exe. No Python installation is required.

FIRST RUN DATA
--------------
The production executable stores writable application files under:

  %LOCALAPPDATA%\EduBidInsight

The settings, database, logs, imports, and backups are created automatically.
The config, data, and backups directories shipped beside the executable are
clean deployment/reference directories and contain no customer information.

BACKUP
------
Use Settings > Backup to create or restore a verified backup. A verified backup
is also created automatically after a normal application exit.

REPORTS
-------
Report Center supports PDF, Excel, and CSV output. Choose Print Preview before
exporting a report.

SUPPORT
-------
Use Settings > About > Run Diagnostics and review the logs under the local
application data directory.
